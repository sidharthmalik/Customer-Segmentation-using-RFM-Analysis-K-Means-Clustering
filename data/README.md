Dataset placement instructions

1. Download the raw dataset from Kaggle:
   https://www.kaggle.com/datasets/sidharthxmalik/customer-segmentation-retail-data
2. Place the Excel file in this folder with the name:
   `online_retail_ii.xlsx`
3. If the file name differs, update the path in `Customer_Segmentation_RFM_KMeans.ipynb` or `customer_segmentation_analysis.py`.

Note: The actual dataset file is not included in this repository to keep the project lightweight and reproducible.
